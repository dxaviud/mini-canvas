# mini-canvas

## About
A simple paint program using Tkinter.
Important keys:
- c: clear canvas
- r: red
- g: greed
- b: blue
- y: yellow
- s: square brush
- o: oval brush
- 0-9: change brush thickness
- =: save canvas as png
- anything else: black  

![minicanvas](https://dxaviud.github.io/images/minicanvas.gif)

## Execute
Install [Pillow](https://pillow.readthedocs.io/en/stable/installation.html)  
`python draw.py`
